Tecnología en Aplicaciones Populares

Lenguajes, Frameworks y País de Origen

TikTok

Lenguajes y tecnologías utilizadas: Java, Kotlin, Swift, JavaScript.
Frameworks y bibliotecas: Utiliza tecnologías propias y de terceros para procesamiento de video y recomendaciones.
Año de lanzamiento: 2016 (como Douyin en China); 2018 (como Tik Tok internacionalmente).
País de origen: China.

Netflix

Lenguajes y tecnologías utilizadas: Java, JavaScript (con React), Python.
Frameworks y bibliotecas: React.
Año de lanzamiento: 1997 (como servicio de alquiler de DVD); 2007 (como servicio de streaming).
País de origen: Estados Unidos.

Apple Music

Lenguajes y tecnologías utilizadas: Swift, Objective-C (para iOS y macOS), Java (para Android), JavaScript (para la interfaz web), Python y C++ (para backend y procesamiento de audio).
Frameworks y bibliotecas: UIKit (para iOS), SwiftUI, Core Audio (para procesamiento de sonido), WebKit (para la versión web).
Año de lanzamiento: 2015.
País de origen: Estados Unidos.

WhatsApp

Lenguajes y tecnologías utilizadas: Erlang, JavaScript (para WhatsApp Web), Java (para Android), Swift (para iOS).
Frameworks y bibliotecas: Utiliza tecnologías propias para mensajería en tiempo real.
Año de lanzamiento: 2009.
País de origen: Estados Unidos.
